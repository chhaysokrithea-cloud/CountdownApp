# ⏳ CountDown Timer

A simple countdown timer application built to track time accurately from Years down to Seconds.

## ✨ Features

- Set countdown duration using:
  - Years
  - Months
  - Weeks
  - Days
  - Hours
  - Minutes
  - Seconds

- Timer Controls:
  - ▶️ Start Timer
  - ⏸️ Stop/Pause Timer
  - 🔄 Clear/Reset Timer

- Real-time countdown display
- Easy and user-friendly interface
- Accurate time calculation

## 🚀 How It Works

1. Enter the desired duration.
2. Click **Start** to begin the countdown.
3. Click **Stop** to pause the timer.
4. Click **Clear** to reset all values.
5. The timer will automatically finish when the countdown reaches zero.

## 🛠️ Technologies Used

- Framework : ReactNative
- Language : Javascript
